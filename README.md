# Episode Info — IINA Plugin

Search for episode or movie info from TMDB and display it as an 
overlay when you pause. Works with any local video file.

## Installation

### From GitHub (recommended — gets auto-updates)
1. Open IINA → Preferences → Plugins
2. Click **Install from GitHub...**
3. Enter: `Zain-Imam/iina-episode-info`
4. Click Install

### Manual
Download the latest `.iinaplgz` from [Releases](../../releases) 
and double-click it.

## Setup
1. Get a free API key at [themoviedb.org](https://www.themoviedb.org) 
   → Settings → API → Developer
2. Open the **Episode Info** tab in IINA's sidebar
3. Paste your key and click Save

## Usage
1. Open the Episode Info sidebar tab
2. Search for your show or movie
3. Pick the episode
4. Pause the video — info appears after 3 seconds
5. Resume — overlay disappears

## Privacy
This plugin only contacts api.themoviedb.org and image.tmdb.org. 
Your API key is stored locally and never shared. No analytics.

## Requirements
- IINA 1.4.0 or later
- macOS 12 or later
- Free TMDB API key
